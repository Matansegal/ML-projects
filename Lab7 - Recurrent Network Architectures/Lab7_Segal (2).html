Methods of Tokenization¶

in order to clear the twits I will remove all the digits, special characters, tagged users and the stop words. I will calculate the average number of words per twits and force the length of sequence to about 50% more than the average, to have twits which are a little longer but also not too long twits.

In [7]:
import re
from nltk import word_tokenize
from nltk.corpus import stopwords

# cleaning all the twits
# and calculating average number of words per twit

stop_words = stopwords.words('english')

words_list = []
tot_len = 0
for twit in df_clear.text:
    # lower case
    item = twit.lower()
    # remove digits
    item = re.sub(r'\d+', ' ', item)
    # remove special chars except @ 
    item = re.sub(r"[-()\"#/;:<>{}`+=~|&.!?,]", "", item)
    # remove tagged users:
    item = re.sub(r'@\w+\s?','',item)
    # split the twit 
    item = word_tokenize(item)
    # removing all stop words
    for w in item:
        if w in stop_words:
            item.remove(w)
            
    words_list.append(item)
    tot_len += len(item)
    
print('avg number of words per twit:',tot_len/len(words_list))

from sklearn import metrics as mt
from sklearn.metrics import roc_curve
from sklearn.metrics import auc
import seaborn as sns
import matplotlib.pyplot as plt
%matplotlib inline

def summarize_net(net, X_test, y_test, title_text=''):
    yhat = net.predict(X_test).round()
    acc = mt.accuracy_score(y_test,yhat)
    cm = mt.confusion_matrix(y_test,yhat)
    cm = cm/np.sum(cm,axis=1)[:,np.newaxis]
    cm = pd.DataFrame(cm , index = ['0','1'] , columns = ['0','1'])
    
    plt.figure(figsize = (12,6))
    plt.subplot(1, 2, 1)
    sns.heatmap(cm,cmap= "coolwarm", linecolor = 'black' , 
                linewidth = 1 , annot = True, fmt='.2f')
    plt.title(title_text+'{:.4f}'.format(acc))
    
    fpr, tpr, thresholds = roc_curve(y_test, yhat)
    auc_keras = auc(fpr, tpr)
    
    plt.subplot(1, 2, 2)
    plt.plot([0, 1], [0, 1], 'k--')
    plt.plot(fpr, tpr, label=' (area = {:.3f})'.format(auc_keras))
    plt.xlabel('False positive rate')
    plt.ylabel('True positive rate')
    plt.title('ROC curve')
    plt.legend(loc='best')
    plt.show()

Comparing between Model 1 and Model 2¶

From the accuracy of the two models we can see that the first model, where I used LSTM with no any pre-processing (77.45%), scored higher than the second model where I used the GRU and the GloVe embedding layer (71.22%). We can also see that the area under the ROC curve of the first model is larger than this of the second model. Even tough I did not accomplish my goal of reaching the 90% accuracy like I said at the beginning, with the first model I reached higher accuracy than I had before, in the data structures class. However, the second model scored lower than before.

I will take the comparison a little farther, using the the McNemars statistic test. My null hypothesis will be that model 2 and the MLP model are the same. In order to reject it with a confidence of 90%, I will need a chi square value higher than 2.706. Once it will be rejected, I will be able to tell with 90% confidence that the two models are different from one another, means that model 2 is better than the MLP model.

# return a bool if we 90% confident to reject the null hypothesis
def get_chi_square(net1,net2,X_test,y_test):
    yhat1 = net1.predict(X_test).round().reshape(y_test.shape)
    yhat2 = net2.predict(X_test).round().reshape(y_test.shape)
    yhat1_acc = yhat1 == y_test
    yhat2_acc = yhat2 == y_test
    b = 0 # 1 correct, 2 not
    c = 0 # 2 correct, 1 not
    
    for i in range(len(y_test)):
        if yhat1_acc[i] and not yhat2_acc[i]:
            b += 1
        elif not yhat1_acc[i] and yhat2_acc[i]:
            c += 1
            
    chi_square = (abs(b-c)-1)**2/(b+c)
    return chi_square > 2.706
