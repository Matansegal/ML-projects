In this lab I will use a data set of pictures of chest x ray images from people with pneumonia and people without. In order to eveluate the algorithms performance I will use the confusion matrix of my algorithm output and the real values. Since we are handling with medical diagnosis, the case of false negative (where we diagnosis a person is healthy where he is actually not) is worse than false positive because it is better to be on the safe side rather than missing a person with pneumonia. Therefore, I will use the following weight metrix for evaluation:

pred 0 pred 1
real 0 0 1
real 1 3 0

when I will have the output of the model I will multiply the confusion matrix with the weight matrix and sum all the element of the product mertix. The lower the sum will be, the better the model is.

from PIL import Image
from resizeimage import resizeimage

# a function to return grey in case of rgb
def rgb2gray(rgb):
    r, g, b = rgb[:,:,0], rgb[:,:,1], rgb[:,:,2]
    gray = 0.2989 * r + 0.5870 * g + 0.1140 * b
    return gray


def resize(img, asnumpy = True):
    with open(img, 'r+b') as f:
        with Image.open(f) as image:
            cover = resizeimage.resize_cover(image, [100, 100], validate=False)
            ret_val = np.array(cover)
            if len(ret_val.shape) == 3:     # if rgb convert to grey
                ret_val = rgb2gray(ret_val)
            return ret_val if asnumpy else cover

from sklearn import metrics as mt
import seaborn as sns
import pandas as pd
%matplotlib inline

def summarize_net(net, X_test, y_test, title_text=''):
    yhat = net.predict(X_test).round()
    acc = mt.accuracy_score(y_test,yhat)
    cm = mt.confusion_matrix(y_test,yhat)
    eval_weights = np.array([[0,1],[3,0]])
    eval_val = np.sum(cm * eval_weights)
    cm = cm/np.sum(cm,axis=1)[:,np.newaxis]
    print('Evaluation value according to the metrix shown at the beginning:',eval_val)
    cm = pd.DataFrame(cm , index = ['0','1'] , columns = ['0','1'])
    plt.figure(figsize = (6,6))
    sns.heatmap(cm,cmap= "coolwarm", linecolor = 'black' , 
                linewidth = 1 , annot = True, fmt='.2f')
    plt.title(title_text+'{:.4f}'.format(acc))

Comparing Model 2 and the MLP model¶

We can see that the evaluation value of the MLP model (185) is lower than the one of model 1 but higher than the second model, means that the performance of model 2 is better than the MLP model according to the conditions we set at the beginning of this lab.

I will take the comparison a little farther, using the the McNemars statistic test. My null hypothesis will be that model 2 and the MLP model are the same. In order to reject it with a confidence of 90%, I will need a chi square value higher than 2.706. Once it will be rejected, I will be able to tell with 90% confidance that the two models are different from one another, means that model 2 is better than the MLP model.

In [87]:
# return a bool if we 90% confident to reject the null hypothesis
def get_chi_square(net1,net2,X_test1,X_test2,y_test):
    yhat1 = net1.predict(X_test1).round().reshape(y_test.shape)
    yhat2 = net2.predict(X_test2).round().reshape(y_test.shape)
    yhat1_acc = yhat1 == y_test
    yhat2_acc = yhat2 == y_test
    b = 0 # 1 correct, 2 not
    c = 0 # 2 correct, 1 not
    
    for i in range(len(y_test)):
        if yhat1_acc[i] and not yhat2_acc[i]:
            b += 1
        elif not yhat1_acc[i] and yhat2_acc[i]:
            c += 1
            
    chi_square = (abs(b-c)-1)**2/(b+c)
    return chi_square > 2.706
